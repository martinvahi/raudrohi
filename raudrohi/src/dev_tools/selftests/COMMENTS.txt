===========================================================================
============  Raudrohi JavaScript Library (RJSL) Selftests ================
===========================================================================

The RJSL project Rakefile task "b_selftests" generates

    ./index.html
    ./bonnet/var/raudrohi_selftests_collection.js

The
    ./bonnet/var/raudrohi_selftests_collection.js

contains the main function that executes the selftests and displays results.

Selftests and the RJSL can be built periodically by

    ruby -Ku ./../pseudo_cron.rb --loop

---------------------------------------------------------------------------
                        (Some of the) Quirks
---------------------------------------------------------------------------

The duplication of the

if (window.raudrohi_my_nice_subnamespace_exists !== true) {
    window.raudrohi.my_nice_subnamespace= {};
    window.raudrohi_my_nice_subnamespace_exists = true;
} // if

and alike is necessary, because the selftests and the
RJSL deliverable reside in different files and the
download order of the files is pseudo-random.

The duplication of

if (window.raudrohi_exists !== true) {
    window.raudrohi = {};
    window.raudrohi_exists = true;
} // if

is NOT necessary, because it is guaranteed to be at
the prefix of the

    ./bonnet/var/raudrohi_selftests_collection.js

The prefix and suffix of the

    ./bonnet/var/raudrohi_selftests_collection.js

are respectively

    ./bonnet/raudrohi_selftests_prefix.js

and
    ./bonnet/raudrohi_selftests_main.js

The JavaScript files that reside in

    ./tests/console

are included to the

    ./bonnet/var/raudrohi_selftests_collection.js

in semi-random order.
